# NAME
  whc - Compile WHILE into CPP

# SYNOPSIS
  java -jar __whc.jar__ `[OPTION]` inputFile

# DESCRIPTION
 Transform a WHILE file into a CPP file.

## Options
  `-o`,
    the output file which will contains the indented while program
